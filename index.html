How it works

1

Capture

Users scan the card (front/back). The system standardizes the input for consistency.

2

Analyze

Pipeline checks the factors that matter to grade outcomes and flags defects with clarity.

3

Explain

Outputs a TruScore + details so the user understands the “why,” not just the number.

4

Decide

Submit confidently or save money. The app’s job is to prevent bad submissions.
